1. Class Hewan

Polimorfisme: objek berbeda menghasilkan suara berbeda.

abstract class Hewan { void suara(); } class Kucing extends Hewan { void suara() => print("Meong"); } class Anjing extends Hewan { void suara() => print("Guk guk"); } void main() { List list = [Kucing(), Anjing()]; list.forEach((h) => h.suara()); }

2. Class Bentuk

Polimorfisme method gambar().

class Bentuk { void gambar() => print("Bentuk"); } class Lingkaran extends Bentuk { void gambar() => print("Lingkaran"); } class Persegi extends Bentuk { void gambar() => print("Persegi"); }

3. Class Kendaraan

Operator is & as untuk pengecekan tipe.

class Kendaraan {} class Mobil extends Kendaraan { void klakson() => print("Tin!"); }

4. Abstract Kendaraan

Abstraksi sebagai dasar class.

abstract class Kendaraan { void jalan(); } class Mobil extends Kendaraan { void jalan() => print("Mobil jalan"); }

5. Class Alat Musik

Abstract class dengan method konkret.

abstract class AlatMusik { void mainkan(); } class Gitar extends AlatMusik { void mainkan() => print("Jreng"); }

6. Class Pesawat

Implements interface.

abstract class Terbang { void terbang(); } class Pesawat implements Terbang { void terbang() => print("Terbang"); }

7. Class MathUtils

Static method.

class MathUtils { static int tambah(int a, int b) => a + b; }

8. Metode Pembayaran

Interface pembayaran.

abstract class Bayar { void proses(); }

9. Bentuk Geometri

Hitung luas.
