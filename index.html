Object Oriented Programming (OOP)

1. Pengertian Polimorfisme

Polimorfisme adalah konsep OOP dimana satu method dapat memiliki banyak bentuk atau implementasi berbeda. Biasanya menggunakan method overriding.

2. Pengertian Abstraksi

Abstraksi adalah proses menyembunyikan detail implementasi dan hanya menampilkan fungsi utama kepada pengguna.

Kesimpulan

Polimorfisme memungkinkan satu method memiliki banyak bentuk sehingga program menjadi fleksibel. Sedangkan abstraksi digunakan untuk menyembunyikan detail program dan hanya menampilkan fungsi pentingnya saja. Dengan menggunakan Polimorfisme dan Abstraksi, program menjadi lebih rapi, mudah dikembangkan, dan mudah dipelihara. Konsep ini sangat penting dalam Object Oriented Programming karena membantu membuat program yang terstruktur dan reusable.
