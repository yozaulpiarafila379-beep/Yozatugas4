1. Polimorfisme (Method Overriding)

Pengertian: Polimorfisme overriding adalah ketika class anak menimpa method dari class induk sehingga memiliki perilaku berbeda.

class Hewan {
  void suara() {
    print("Hewan bersuara");
  }
}

class Kucing extends Hewan {
  @override
  void suara() {
    print("Kucing mengeong");
  }
}

void main() {
  Hewan h = Kucing();
  h.suara();
}

2. Named Constructor

Pengertian: Named constructor adalah constructor yang memiliki nama sehingga satu class bisa memiliki lebih dari satu constructor.

class Mahasiswa {
  String nama;
  int umur;

  Mahasiswa(this.nama, this.umur);

  Mahasiswa.lulus(this.nama){
    umur = 22;
  }
}

void main() {
  Mahasiswa m = Mahasiswa.lulus("Yoza");
  print(m.nama);
  print(m.umur);
}

3. Abstraksi (Abstract Class)

Pengertian: Abstraksi adalah proses menyembunyikan detail implementasi dan hanya menampilkan fungsi utama.

abstract class Kendaraan {
  void jalan();
}

class Mobil extends Kendaraan {
  void jalan() {
    print("Mobil berjalan");
  }
}

void main() {
  Mobil m = Mobil();
  m.jalan();
}

4. Interface

Pengertian: Interface adalah class yang harus diimplementasikan oleh class lain.

abstract class Bentuk {
  void gambar();
}

class Lingkaran implements Bentuk {
  void gambar() {
    print("Gambar Lingkaran");
  }
}

void main() {
  Lingkaran l = Lingkaran();
  l.gambar();
}

5. Inheritance

Pengertian: Inheritance adalah pewarisan sifat dari class induk ke class anak.

class Hewan {
  String nama = "Kucing";
}

class Kucing extends Hewan {
  void tampil() {
    print(nama);
  }
}

void main() {
  Kucing k = Kucing();
  k.tampil();
}

6. Optional Parameter

Pengertian: Optional parameter adalah parameter yang tidak wajib diisi saat pemanggilan method.

class Hitung {
  int tambah(int a, int b, [int c = 0]) {
    return a + b + c;
  }
}

void main() {
  Hitung h = Hitung();
  print(h.tambah(2,3));
  print(h.tambah(2,3,4));
}

7. Encapsulation (Getter & Setter)

Pengertian: Encapsulation adalah menyembunyikan data menggunakan getter dan setter.

class Mahasiswa {
  String _nama;

  void setNama(String nama){
    _nama = nama;
  }

  String getNama(){
    return _nama;
  }
}

void main() {
  Mahasiswa m = Mahasiswa();
  m.setNama("Yoza");
  print(m.getNama());
}

8. Abstract Class

Pengertian: Abstract class adalah class yang tidak bisa dibuat objek langsung.

abstract class Hewan {
  void suara();
}

class Sapi extends Hewan {
  void suara(){
    print("Moo");
  }
}

void main() {
  Sapi s = Sapi();
  s.suara();
}

9. Polimorfisme

Pengertian: Polimorfisme adalah satu method memiliki banyak bentuk.

class Hewan {
  void suara() {
    print("Hewan bersuara");
  }
}

class Ayam extends Hewan {
  void suara() {
    print("Kukuruyuk");
  }
}

void main() {
  Hewan h = Ayam();
  h.suara();
}

10. Static Method

Pengertian: Static adalah method atau variabel yang bisa dipanggil tanpa membuat objek.

class Matematika {
  static int tambah(int a, int b) {
    return a + b;
  }
}

void main() {
  print(Matematika.tambah(2,3));
}

  Mahasiswa m = Mahasiswa("Yoza");

14. Method Luas Persegi

class Persegi {
  int sisi;

  Persegi(this.sisi);

  int luas(){
    return sisi * sisi;
  }
}

void main() {
  Persegi p = Persegi(4);
  print(p.luas());
}
