Kesimpulan

Polimorfisme adalah kemampuan satu method memiliki banyak bentuk. Abstraksi adalah menyembunyikan detail program. Inheritance adalah pewarisan dari class induk ke class anak. Interface adalah kontrak method. Static digunakan agar method dapat dipanggil tanpa objek. OOP membuat program lebih rapi dan mudah dikembangkan.
