9. Class Bentuk Geometri

Polimorfisme menghitung luas dan keliling.

abstract class Bentuk { double hitungLuas(); double hitungKeliling(); void info() { print('Luas: ${hitungLuas()}, Keliling: ${hitungKeliling()}'); } } class Persegi extends Bentuk { double sisi; Persegi(this.sisi); double hitungLuas() => sisi * sisi; double hitungKeliling() => 4 * sisi; } class Lingkaran extends Bentuk { double radius; Lingkaran(this.radius); double hitungLuas() => 3.14 * radius * radius; double hitungKeliling() => 2 * 3.14 * radius; } void main() { Persegi p = Persegi(4); Lingkaran l = Lingkaran(7); p.info(); l.info(); }

10. Class Programmer

Inheritance dari class Pekerja.

abstract class Pekerja { String nama; Pekerja(this.nama); void bekerja(); } class Programmer extends Pekerja { Programmer(String nama) : super(nama); void bekerja() { print("$nama bekerja sebagai Programmer"); } } void main() { Programmer p = Programmer("Andi"); p.bekerja(); }

11. Multi Class Pekerja

Polimorfisme banyak class turunan.

abstract class Pekerja { String nama; Pekerja(this.nama); void bekerja(); } class Programmer extends Pekerja { Programmer(String nama) : super(nama); void bekerja() => print("$nama adalah Programmer"); } class Dokter extends Pekerja { Dokter(String nama) : super(nama); void bekerja() => print("$nama adalah Dokter"); } class Guru extends Pekerja { Guru(String nama) : super(nama); void bekerja() => print("$nama adalah Guru"); } void main() { Programmer p = Programmer("Andi"); Dokter d = Dokter("Budi"); Guru g = Guru("Citra"); p.bekerja(); d.bekerja(); g.bekerja(); }

12. Polimorfisme List

List berisi banyak object berbeda (polimorfisme).

abstract class Pekerja { String nama; Pekerja(this.nama); void bekerja(); } class Programmer extends Pekerja { Programmer(String nama) : super(nama); void bekerja() => print("$nama sedang coding"); } class Dokter extends Pekerja { Dokter(String nama) : super(nama); void bekerja() => print("$nama sedang mengobati pasien"); } class Guru extends Pekerja { Guru(String nama) : super(nama); void bekerja() => print("$nama sedang mengajar"); } void main() { List daftar = [ Programmer("Andi"), Dokter("Budi"), Guru("Citra"), ]; for (var p in daftar) { p.bekerja(); } }

13. Class Tablet

Multiple interface.

abstract class Komputer { void menjalankanProgram(); } abstract class Kamera { void ambilFoto(); } abstract class Telepon { void menelepon(); } class Tablet implements Komputer, Kamera, Telepon { void menjalankanProgram() { print("Tablet menjalankan program"); } void ambilFoto() { print("Tablet mengambil foto"); } void menelepon() { print("Tablet menelepon"); } } void main() { Tablet tablet = Tablet(); tablet.menjalankanProgram(); tablet.ambilFoto(); tablet.menelepon(); }

14. MathUtils Advanced

Static method matematika.

import 'dart:math'; class MathUtils { static int faktorial(int n) { if (n <= 1) return 1; return n * faktorial(n - 1); } static bool cekPrima(int n) { if (n <= 1) return false; for (int i = 2; i <= sqrt(n); i++) { if (n % i == 0) return false; } return true; } static double rataRata(List list) { int total = list.reduce((a, b) => a + b); return total / list.length; } } void main() { print(MathUtils.faktorial(5)); print(MathUtils.cekPrima(7)); print(MathUtils.rataRata([10,20,30,40])); }

Kesimpulan

Polimorfisme adalah kemampuan method memiliki banyak bentuk, sedangkan abstraksi adalah proses menyembunyikan detail implementasi. Inheritance adalah pewarisan dari class induk ke class anak, interface digunakan untuk membuat standar method, dan static digunakan agar method dapat dipanggil tanpa membuat objek. Dengan OOP, program menjadi lebih rapi, mudah dikembangkan, dan reusable.
