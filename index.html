1. Polimorfisme Overriding

Overriding adalah method di class anak yang menggantikan method dari class induk.

class Hewan {
  void suara() { print("Hewan bersuara"); }
}
class Kucing extends Hewan {
  void suara() { print("Kucing mengeong"); }
}
void main() {
  Hewan h = Kucing();
  h.suara();
}

2. Named Constructor

Constructor dengan nama agar bisa membuat objek dengan cara berbeda.

class Mahasiswa {
  String nama;
  int umur;

  Mahasiswa(this.nama, this.umur);
  Mahasiswa.lulus(this.nama){
    umur = 22;
  }
}

3. Abstraksi

Menyembunyikan detail dan hanya menampilkan fungsi penting.

abstract class Kendaraan {
  void jalan();
}
class Mobil extends Kendaraan {
  void jalan() {
    print("Mobil berjalan");
  }
}

4. Interface

Class yang harus diimplementasikan oleh class lain.

abstract class Bentuk {
  void gambar();
}
class Lingkaran implements Bentuk {
  void gambar() {
    print("Gambar Lingkaran");
  }
}

5. Inheritance

Pewarisan sifat dari class induk ke class anak.

class Hewan {
  String nama = "Kucing";
}
class Kucing extends Hewan {
  void tampil() {
    print(nama);
  }
}

6. Overloading

Method dengan parameter berbeda dalam satu fungsi.

class Hitung {
  int tambah(int a, int b, [int c = 0]) {
    return a + b + c;
  }
}

7. Encapsulation

Menyembunyikan data menggunakan getter dan setter.

class Mahasiswa {
  String _nama;

  void setNama(String nama){
    _nama = nama;
  }

  String getNama(){
    return _nama;
  }
}

8. Abstract Class

Class yang tidak bisa dibuat objek.

abstract class Hewan {
  void suara();
}
class Sapi extends Hewan {
  void suara(){
    print("Moo");
  }
}

9. Polimorfisme

Satu method memiliki banyak bentuk.

class Hewan {
  void suara() { print("Hewan bersuara"); }
}
class Ayam extends Hewan {
  void suara() { print("Kukuruyuk"); }
}

10. Static

Method bisa dipanggil tanpa objek.

class Matematika {
  static int tambah(int a, int b) {
    return a + b;
  }
}

11. Class & Object

Class adalah cetakan, object adalah hasilnya.

class Mobil {
  String merk = "Toyota";
}

12. Constructor

Dijalankan saat objek dibuat.

class Mahasiswa {
  String nama;
  Mahasiswa(this.nama);
}

13. This Keyword

Digunakan untuk menunjuk objek saat ini.

class Mahasiswa {
  String nama;
  Mahasiswa(String nama){
    this.nama = nama;
  }
}

14. Luas Persegi

Program OOP untuk menghitung luas.

class Persegi {
  int sisi;
  Persegi(this.sisi);

  int luas(){
    return sisi * sisi;
  }
}

Kesimpulan

Konsep OOP seperti Polimorfisme, Abstraksi, Inheritance, dan Encapsulation membuat program lebih terstruktur, mudah dipahami, dan dapat digunakan kembali.
