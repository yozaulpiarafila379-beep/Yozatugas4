1. Class Hewan

Pengertian: Polimorfisme adalah kemampuan satu method memiliki banyak bentuk. Pada contoh ini setiap class turunan Hewan memiliki method bersuara() yang berbeda.

abstract class Hewan { String nama = "Hewan"; void bersuara(); } class Kucing extends Hewan { Kucing(){ nama="Kucing"; } void bersuara(){ print("$nama: Meong"); } } class Anjing extends Hewan { Anjing(){ nama="Anjing"; } void bersuara(){ print("$nama: Guk guk"); } } class Burung extends Hewan { Burung(){ nama="Burung"; } void bersuara(){ print("$nama: Cuit cuit"); } } void main(){ List daftar=[Kucing(),Anjing(),Burung()]; for(var h in daftar){ h.bersuara(); } }

2. Class Bentuk

Pengertian: Polimorfisme method overriding, yaitu method yang sama tetapi memiliki implementasi berbeda pada class turunan.

class Bentuk { void gambar(){ print("Menggambar bentuk"); } } class Lingkaran extends Bentuk { void gambar(){ print("Menggambar lingkaran"); } } class Persegi extends Bentuk { void gambar(){ print("Menggambar persegi"); } } void main(){ List list=[Lingkaran(),Persegi()]; for(var b in list){ b.gambar(); } }

3. Class Kendaraan

Pengertian: Operator is digunakan untuk mengecek tipe object dan as digunakan untuk casting tipe object.

class Kendaraan {} class Mobil extends Kendaraan { void klakson(){ print("Tin!"); } } void main(){ Kendaraan k=Mobil(); if(k is Mobil){ k.klakson(); } (k as Mobil?)?.klakson(); }

4. Abstract Kendaraan

Pengertian: Abstraksi adalah proses menyembunyikan detail dan hanya menampilkan fungsi utama. Abstract class tidak bisa dibuat objek.

abstract class Kendaraan{ void jalan(); } class Mobil extends Kendaraan{ void jalan(){ print("Mobil berjalan"); } } void main(){ Mobil m=Mobil(); m.jalan(); }

5. Class AlatMusik

Pengertian: Abstract class dapat memiliki method abstract dan method biasa (concrete).

abstract class AlatMusik{ String nama; AlatMusik(this.nama); void mainkan(); void info(){ print("Alat musik $nama"); } } class Gitar extends AlatMusik{ Gitar(String nama):super(nama); void mainkan(){ print("Gitar berbunyi"); } } void main(){ Gitar g=Gitar("Akustik"); g.info(); g.mainkan(); }

6. Class Pesawat

Pengertian: Interface digunakan untuk membuat kontrak method yang harus diimplementasikan oleh class lain.

abstract class Kendaraan{ void jalan(); } abstract class Terbang{ void terbang(); } class Pesawat extends Kendaraan implements Terbang{ void jalan(){ print("Pesawat berjalan"); } void terbang(){ print("Pesawat terbang"); } } void main(){ Pesawat p=Pesawat(); p.jalan(); p.terbang(); }

7. MathUtils

Pengertian: Static digunakan agar method atau variabel bisa dipanggil tanpa membuat objek.

class MathUtils{ static double pi=3.14; static int add(int a,int b)=>a+b; } void main(){ print(MathUtils.pi); print(MathUtils.add(10,20)); }

8. Metode Pembayaran

Pengertian: Interface memungkinkan banyak class memiliki method yang sama tetapi implementasi berbeda.

abstract class MetodePembayaran{ void bayar(double jumlah); } class KartuKredit implements MetodePembayaran{ void bayar(double jumlah){ print("Bayar kartu kredit"); } } class EWallet implements MetodePembayaran{ void bayar(double jumlah){ print("Bayar ewallet"); } } void main(){ KartuKredit k=KartuKredit(); EWallet e=EWallet(); k.bayar(10000); e.bayar(5000); }

Kesimpulan

Polimorfisme adalah kemampuan satu method memiliki banyak bentuk. Abstraksi adalah proses menyembunyikan detail program dan hanya menampilkan fungsi penting. Inheritance adalah pewarisan dari class induk ke class anak. Interface adalah kontrak method yang harus diimplementasikan. Static digunakan agar method bisa dipanggil tanpa objek. Dengan konsep OOP, program menjadi lebih terstruktur, mudah dikembangkan, dan dapat digunakan kembali.
