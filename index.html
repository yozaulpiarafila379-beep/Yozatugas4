1. Class Hewan (Polimorfisme)

Polimorfisme dimana setiap turunan menghasilkan suara berbeda.

abstract class Hewan { String nama = "Hewan"; void bersuara(); } class Kucing extends Hewan { Kucing() { nama = "Kucing"; } void bersuara() { print('$nama: Meong'); } } class Anjing extends Hewan { Anjing() { nama = "Anjing"; } void bersuara() { print('$nama: Guk guk'); } } class Burung extends Hewan { Burung() { nama = "Burung"; } void bersuara() { print('$nama: Cuit cuit'); } } void main() { List daftar = [Kucing(), Anjing(), Burung()]; for (var h in daftar) { h.bersuara(); } }

2. Class Bentuk (Polimorfisme)

Polimorfisme method gambar().

class Bentuk { void gambar() { print('Menggambar bentuk'); } } class Lingkaran extends Bentuk { void gambar() { print('Menggambar lingkaran'); } } class Persegi extends Bentuk { void gambar() { print('Menggambar persegi'); } } void main() { List daftar = [Lingkaran(), Persegi()]; for (var b in daftar) { b.gambar(); } }

3. Class Kendaraan (Operator is & as)

Mengecek tipe objek dan casting.

class Kendaraan {} class Mobil extends Kendaraan { void klakson() { print('Tin!'); } } void main() { Kendaraan k = Mobil(); if (k is Mobil) { k.klakson(); } (k as Mobil?)?.klakson(); }

4. Abstract Class Kendaraan

Abstraksi sebagai dasar class turunan.

abstract class Kendaraan { void jalan(); } class Mobil extends Kendaraan { void jalan() { print("Mobil berjalan"); } } void main() { Mobil m = Mobil(); m.jalan(); }

5. Class AlatMusik (Abstraksi)

Abstract class dengan method abstract dan concrete.

abstract class AlatMusik { String nama; AlatMusik(this.nama); void mainkan(); void info() { print('Ini adalah alat musik $nama'); } } class Gitar extends AlatMusik { Gitar(String nama) : super(nama); void mainkan() { print('Gitar $nama berbunyi: Jreng jreng...'); } } void main() { Gitar gitar = Gitar("Akustik"); gitar.info(); gitar.mainkan(); }

6. Class Pesawat (Interface)

Implementasi lebih dari satu interface.

abstract class Kendaraan { void jalan(); } abstract class Terbang { void terbang(); } class Pesawat extends Kendaraan implements Terbang { void jalan() { print('Pesawat berjalan di landasan'); } void terbang() { print('Pesawat terbang di udara'); } } void main() { Pesawat pesawat = Pesawat(); pesawat.jalan(); pesawat.terbang(); }

7. Class MathUtils (Static)

Static variable dan method.

class MathUtils { static double pi = 3.14; static int add(int a, int b) { return a + b; } } void main() { print(MathUtils.pi); print(MathUtils.add(10, 20)); }

8. Class MetodePembayaran (Interface)

Interface metode pembayaran.

abstract class MetodePembayaran { void bayar(double jumlah); } class KartuKredit implements MetodePembayaran { void bayar(double jumlah) { print('Membayar Rp$jumlah menggunakan Kartu Kredit'); } } class EWallet implements MetodePembayaran { void bayar(double jumlah) { print('Membayar Rp$jumlah menggunakan E-Wallet'); } } void main() { KartuKredit kartu = KartuKredit(); EWallet ewallet = EWallet(); kartu.bayar(50000); ewallet.bayar(25000); }

10. Class Programmer

Inheritance dari Pekerja.

abstract class Pekerja { String nama; Pekerja(this.nama); void bekerja(); } class Programmer extends Pekerja { Programmer(String nama) : super(nama); void bekerja() { print("$nama bekerja sebagai Programmer"); } } void main() { Programmer p = Programmer("Andi"); p.bekerja(); }

Kesimpulan

Polimorfisme adalah kemampuan method memiliki banyak bentuk, sedangkan abstraksi adalah proses menyembunyikan detail implementasi. Dengan konsep OOP seperti inheritance, interface, dan static, program menjadi lebih terstruktur, mudah dikembangkan, dan reusable.
