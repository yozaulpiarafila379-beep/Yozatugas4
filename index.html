1. Class Hewan

Program ini menjelaskan konsep polimorfisme dimana method bersuara() memiliki banyak bentuk pada class Kucing, Anjing, dan Burung.

abstract class Hewan { void bersuara(); } class Kucing extends Hewan { void bersuara() => print("Meong"); } class Anjing extends Hewan { void bersuara() => print("Guk guk"); } void main() { List<Hewan> hewan = [Kucing(), Anjing()]; for (var h in hewan) { h.bersuara(); } }

2. Class Bentuk

Program ini menjelaskan method overriding dimana class turunan mengubah method dari class induk.

class Bentuk { void gambar() { print("Menggambar bentuk"); } } class Lingkaran extends Bentuk { void gambar() => print("Menggambar Lingkaran"); } class Persegi extends Bentuk { void gambar() => print("Menggambar Persegi"); }

3. Class Kendaraan (is & as)

Program ini menjelaskan penggunaan operator is dan as untuk mengecek tipe object.

class Kendaraan {} class Mobil extends Kendaraan { void klakson() { print("Tin!"); } } void main() { Kendaraan k = Mobil(); if (k is Mobil) { (k as Mobil).klakson(); } }

4. Abstract Class Kendaraan

Program ini menjelaskan abstraksi dimana class abstract memiliki method yang harus diimplementasikan oleh class turunan.

abstract class Kendaraan { void start(); } class Mobil extends Kendaraan { void start() { print("Mobil menyala"); } }

5. Class Alat Musik

Program ini menjelaskan inheritance dimana class Gitar mewarisi class AlatMusik.

abstract class AlatMusik { void mainkan(); } class Gitar extends AlatMusik { void mainkan() { print("Jreng..."); } }

6. Interface Pesawat

Program ini menjelaskan multiple interface.

abstract class Terbang { void terbang(); } class Pesawat implements Terbang { void terbang() { print("Pesawat terbang"); } }

7. Static MathUtils

Program ini menjelaskan static method.

class MathUtils { static int tambah(int a, int b) { return a + b; } } void main() { print(MathUtils.tambah(5, 3)); }

8. Metode Pembayaran

Program ini menjelaskan interface dan polimorfisme.

abstract class Pembayaran { void bayar(); } class EWallet implements Pembayaran { void bayar() => print("Bayar dengan E-Wallet"); }

Kesimpulan

Polimorfisme adalah kemampuan method memiliki banyak bentuk. Abstraksi adalah proses menyembunyikan detail implementasi. Inheritance adalah pewarisan dari class induk ke class anak. Interface digunakan untuk membuat standar method. Static digunakan agar method dapat dipanggil tanpa objek. Dengan OOP program menjadi lebih rapi dan mudah dikembangkan.
