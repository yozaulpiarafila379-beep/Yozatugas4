1. Class Hewan

Polimorfisme adalah kemampuan method memiliki banyak bentuk. Class Hewan memiliki method suara() yang di-override oleh Kucing dan Anjing.
class Hewan { void suara() { print("Hewan bersuara"); } } class Kucing extends Hewan { @override void suara() { print("Meong"); } } class Anjing extends Hewan { @override void suara() { print("Guk guk"); } }

2. Class Bentuk

Polimorfisme method gambar() pada Lingkaran dan Persegi.
class Bentuk { void gambar() { print("Menggambar bentuk"); } } class Lingkaran extends Bentuk { void gambar() { print("Menggambar Lingkaran"); } } class Persegi extends Bentuk { void gambar() { print("Menggambar Persegi"); } }

3. Operator is dan as

Operator is untuk mengecek tipe object, as untuk casting tipe object.
Kendaraan k = Mobil(); if (k is Mobil) { k.klakson(); } (k as Mobil?)?.klakson();

4. Abstract Class Kendaraan

Abstract class tidak dapat dibuat objek langsung dan digunakan sebagai dasar class turunan.
abstract class Kendaraan { void start(); } class Mobil extends Kendaraan { void start() { print("Mobil menyala"); } }

5. Abstract Class Alat Musik

abstract class AlatMusik { void mainkan(); } class Gitar extends AlatMusik { void mainkan() { print("Jreng..."); } }

6. Interface Pesawat

abstract class Terbang { void terbang(); } class Pesawat implements Terbang { void terbang() { print("Pesawat terbang"); } }

7. Static MathUtils

class MathUtils { static int add(int a, int b) { return a + b; } }

8. Interface MetodePembayaran

abstract class MetodePembayaran { void bayar(double jumlah); }

9. Bentuk Geometri

abstract class Bentuk { double hitungLuas(); double hitungKeliling(); }

10. Inheritance Pekerja

class Pekerja { String nama; Pekerja(this.nama); } class Programmer extends Pekerja { Programmer(String nama) : super(nama); }

11. Polimorfisme Pekerja

class Programmer extends Pekerja { void bekerja() => print("Coding"); } class Dokter extends Pekerja { void bekerja() => print("Mengobati"); }

12. List Polimorfisme

List daftar = [ Programmer("Andi"), Dokter("Budi"), ];

13. Multiple Interface Tablet

class Tablet implements Komputer, Kamera, Telepon { void menjalankanProgram(){} void ambilFoto(){} void menelepon(){} }

14. MathUtils Advanced

class MathUtils { static int faktorial(int n) {...} static bool cekPrima(int n) {...} static double rataRata(List list) {...} }

Kesimpulan

Berdasarkan praktikum yang telah dilakukan, dapat disimpulkan bahwa konsep OOP pada Dart meliputi Class, Inheritance, Polymorphism, Abstract Class, Interface, dan Static Method. Polimorfisme memungkinkan satu method memiliki banyak bentuk, inheritance memungkinkan pewarisan sifat dari class induk ke class anak, abstract class digunakan sebagai kerangka dasar, interface digunakan untuk membuat standar method, dan static digunakan agar method dapat dipanggil tanpa membuat objek. Dengan menggunakan OOP, program menjadi lebih rapi, terstruktur, dan mudah dikembangkan kembali.
