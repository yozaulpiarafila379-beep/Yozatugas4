1. Polimorfisme Overriding

Polimorfisme overriding adalah kemampuan class anak menimpa method dari class induk.

class Hewan {
  void suara() { print("Hewan bersuara"); }
}
class Kucing extends Hewan {
  @override
  void suara() { print("Kucing mengeong"); }
}
void main() {
  Hewan h = Kucing();
  h.suara();
}

2. Named Constructor

Named constructor adalah constructor yang memiliki nama.

class Mahasiswa {
  String nama;
  int umur;
  Mahasiswa(this.nama, this.umur);
  Mahasiswa.lulus(this.nama){ umur = 22; }
}

3. Abstraksi

Abstraksi adalah menyembunyikan detail implementasi.

abstract class Kendaraan {
  void jalan();
}
class Mobil extends Kendaraan {
  void jalan() { print("Mobil berjalan"); }
}

4. Interface

Interface adalah class yang harus diimplementasikan.

abstract class Bentuk {
  void gambar();
}
class Lingkaran implements Bentuk {
  void gambar() { print("Gambar Lingkaran"); }
}

5. Inheritance

Inheritance adalah pewarisan sifat.

class Hewan {
  String nama = "Kucing";
}
class Kucing extends Hewan {
  void tampil() { print(nama); }
}

6. Overloading

Overloading adalah method dengan parameter berbeda.

class Hitung {
  int tambah(int a, int b, [int c = 0]) {
    return a + b + c;
  }
}

7. Encapsulation

Encapsulation adalah menyembunyikan data.

class Mahasiswa {
  String _nama;
  void setNama(String nama){ _nama = nama; }
  String getNama(){ return _nama; }
}

8. Abstract Class

Abstract class tidak bisa dibuat objek.

abstract class Hewan {
  void suara();
}
class Sapi extends Hewan {
  void suara(){ print("Moo"); }
}

9. Polimorfisme

Polimorfisme adalah banyak bentuk dalam satu method.

class Hewan {
  void suara() { print("Hewan bersuara"); }
}
class Ayam extends Hewan {
  void suara() { print("Kukuruyuk"); }
}

10. Static

Static dapat dipanggil tanpa objek.

class Matematika {
  static int tambah(int a, int b) {
    return a + b;
  }
}

11. Class dan Object

Class adalah blueprint, object adalah hasilnya.

class Mobil {
  String merk = "Toyota";
}

12. Constructor

Constructor dijalankan saat objek dibuat.

class Mahasiswa {
  String nama;
  Mahasiswa(this.nama);
}

13. This Keyword

This digunakan untuk menunjuk objek saat ini.

class Mahasiswa {
  String nama;
  Mahasiswa(String nama){
    this.nama = nama;
  }
}

14. Program Luas Persegi

Contoh program OOP menghitung luas persegi.

class Persegi {
  int sisi;
  Persegi(this.sisi);
  int luas(){
    return sisi * sisi;
  }
}

Dari praktikum ini dapat disimpulkan bahwa OOP memiliki banyak konsep seperti Polimorfisme, Abstraksi, Inheritance, Encapsulation, Interface, Constructor, dan lainnya. Dengan OOP program menjadi lebih rapi, mudah dikembangkan, dan dapat digunakan kembali.
