1. Class Hewan (Polimorfisme)

Mendemonstrasikan polimorfisme dan abstraksi. Hewan sebagai kelas induk, Kucing, Anjing, Burung memiliki cara bersuara berbeda.


// ================================
// Nama : Yoza Ulpia Rafila
// NIM        : 251410004
// Kelas : SI2A
// ================================

abstract class Hewan {
  String nama = "Hewan";
  void bersuara();
}

class Kucing extends Hewan {
  Kucing() { nama = "Kucing"; }
  @override void bersuara() { print('$nama: Meong'); }
}

class Anjing extends Hewan {
  Anjing() { nama = "Anjing"; }
  @override void bersuara() { print('$nama: Guk guk'); }
}

class Burung extends Hewan {
  Burung() { nama = "Burung"; }
  @override void bersuara() { print('$nama: Cuit cuit'); }
}

void main() {
  List daftar = [Kucing(), Anjing(), Burung()];
  for (var h in daftar) { h.bersuara(); }
}

2. Class Bentuk (Polimorfisme)

Contoh method overriding. Kelas Bentuk memiliki method gambar() yang diubah cara kerjanya oleh Lingkaran dan Persegi.


// ================================
// Nama : Yoza Ulpia Rafila
// NIM        : 251410004
// Kelas : SI2A
// ================================

class Bentuk {
  void gambar() { print('Menggambar bentuk'); }
}

class Lingkaran extends Bentuk {
  @override void gambar() { print('Menggambar lingkaran'); }
}

class Persegi extends Bentuk {
  @override void gambar() { print('Menggambar persegi'); }
}

void main() {
  List daftar = [Lingkaran(), Persegi()];
  for (var b in daftar) { b.gambar(); }
}

3. Operator is & as

Cara mengecek tipe objek dengan 'is' dan mengubah tipe dengan 'as' agar bisa akses method khusus.


/// ================================
/// Nama : Yoza Ulpia Rafila
/// NIM        : 251410004
/// Kelas : SI2A
/// ================================

class Kendaraan {}

class Mobil extends Kendaraan {
  void klakson() { print('Tin!'); }
}

void main() {
  Kendaraan k = Mobil();

  if (k is Mobil) {
    print("k adalah Mobil");
    k.klakson();
  }

  (k as Mobil?)?.klakson();
}

4. Abstract Class Kendaraan

Kelas abstrak sebagai kerangka dasar. Method start() harus dibuat ulang, method stop() sudah jadi.


/// ================================
/// Nama : Yoza Ulpia Rafila
/// NIM        : 251410004
/// Kelas : SI2A
/// ================================

abstract class Kendaraan {
  String merk;
  Kendaraan(this.merk);
  void start(); // Wajib diisi
  void stop() { print('Kendaraan berhenti'); }
}

class Mobil extends Kendaraan {
  Mobil(String merk) : super(merk);
  @override void start() { print('Mobil $merk menyala'); }
}

void main() {
  Mobil m = Mobil("Toyota");
  m.start();
  m.stop();
}

5. Class AlatMusik

Sama seperti abstraksi sebelumnya, AlatMusik punya method mainkan() yang berbeda tiap alat.


/// ================================
/// Nama : Yoza Ulpia Rafila
/// NIM        : 251410004
/// Kelas : SI2A
/// ================================

abstract class AlatMusik {
  String nama;
  AlatMusik(this.nama);
  void mainkan();
  void info() { print('Ini alat musik $nama'); }
}

class Gitar extends AlatMusik {
  Gitar(String nama) : super(nama);
  @override void mainkan() { print('Gitar $nama: Jreng jreng...'); }
}

void main() {
  Gitar g = Gitar("Akustik");
  g.info();
  g.mainkan();
}

6. Class Pesawat (Interface)

Pesawat mewarisi Kendaraan dan mengimplementasikan Terbang. Harus buat semua method.


/// ================================
/// Nama : Yoza Ulpia Rafila
/// NIM        : 251410004
/// Kelas : SI2A
/// ================================

abstract class Kendaraan { void jalan(); }
abstract class Terbang { void terbang(); }

class Pesawat extends Kendaraan implements Terbang {
  @override void jalan() { print('Pesawat berjalan'); }
  @override void terbang() { print('Pesawat terbang'); }
}

void main() {
  Pesawat p = Pesawat();
  p.jalan();
  p.terbang();
}
