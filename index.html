1. Class Hewan

Program ini menjelaskan konsep polimorfisme dimana method bersuara() memiliki banyak bentuk pada class Kucing, Anjing, dan Burung.

abstract class Hewan { void bersuara(); } class Kucing extends Hewan { void bersuara() => print("Meong"); } class Anjing extends Hewan { void bersuara() => print("Guk guk"); } void main() { List<Hewan> hewan = [Kucing(), Anjing()]; for (var h in hewan) { h.bersuara(); } }

2. Class Bentuk

Program ini menjelaskan method overriding dimana class turunan mengubah isi method dari class induk.

class Bentuk { void gambar() { print("Menggambar bentuk"); } } class Lingkaran extends Bentuk { void gambar() { print("Menggambar Lingkaran"); } }

3. Kendaraan (is & as)

Program ini menjelaskan penggunaan operator is dan as untuk mengecek dan mengubah tipe objek.

class Kendaraan {} class Mobil extends Kendaraan { void klakson() => print("Tin!"); }

4. Abstract Kendaraan

Program ini menjelaskan abstraksi dimana class abstract memiliki method yang harus diimplementasikan oleh class turunan.

abstract class Kendaraan { void start(); }

5. Alat Musik

Program ini menjelaskan inheritance dimana class Gitar mewarisi class AlatMusik.

abstract class AlatMusik { void mainkan(); }

6. Pesawat

Program ini menjelaskan multiple interface dimana class dapat mengimplementasikan lebih dari satu interface.

abstract class Terbang { void terbang(); }

7. MathUtils Static

Program ini menjelaskan static method yang dapat dipanggil tanpa membuat objek.

class MathUtils { static int tambah(int a, int b) => a + b; }

8. Metode Pembayaran

Program ini menjelaskan interface dan polimorfisme pada metode pembayaran.

abstract class Pembayaran { void bayar(); }

Kesimpulan

Polimorfisme adalah kemampuan method memiliki banyak bentuk, sedangkan abstraksi adalah proses menyembunyikan detail implementasi. Inheritance adalah pewarisan dari class induk ke class anak, interface digunakan untuk membuat standar method, dan static digunakan agar method dapat dipanggil tanpa membuat objek. Dengan OOP, program menjadi lebih rapi, mudah dikembangkan, dan reusable.
